# 安装

从 [Releases](https://github.com/vogo/namer/releases) 页面下载对应平台的二进制文件（支持 macOS、Linux、Windows）。

或通过 Go 源码编译安装：

```bash
go install github.com/vogo/namer@latest
```

安装完成后，确保 `$GOPATH/bin` 在 `PATH` 中，即可使用 `namer` 命令。

# 评分维度

总分 100 分：

| 维度 | 分值 | 说明 |
|------|------|------|
| 五格数理 | 30分 | 天格、人格、地格、总格、外格的数理吉凶 |
| 三才配置 | 25分 | 天人地三才的五行生克关系 |
| 喜用神 | 20分 | 名字五行是否补益八字喜用神 |
| 内部五行 | 15分 | 姓名各字之间的五行生克 |
| 阴阳平衡 | 10分 | 姓名各字笔画的阴阳搭配 |

# 使用示例

工具提供两大核心功能：**名字评分**（对指定名字打分）和**批量生成建议名字**（根据备选字自动组合并评分排序）。

## 一、名字评分

对已有名字进行五行评分分析。

### 命令行评分

```bash
# 基本用法（使用已有配置中的生辰信息）
$ namer -xing 王 -ming 明轩

# 指定完整出生信息
$ namer -xing 王 -ming 明轩 -year 2024 -month 3 -day 15 -hour 10 -minute 30

# 指定性别（1=男, 2=女）
$ namer -xing 王 -ming 明轩 -year 2024 -month 3 -day 15 -hour 10 -minute 30 -gender 1

# 单字名评分
$ namer -xing 王 -ming 轩

# 双字名评分
$ namer -xing 王 -ming 明轩

# 复姓评分
$ namer -xing 欧阳 -ming 明轩

# 仅指定部分出生信息（其余从配置文件读取）
$ namer -xing 王 -ming 明轩 -year 2024
```

## 二、批量生成建议名字

根据备选字自动组合（单字名 + 双字名），评分排序后输出高分名字。

### 命令行批量生成

```bash
# 通过命令行参数批量生成（无需配置文件）
$ namer -xing 王 -keywords 明,轩,浩,然 -year 2024 -month 3 -day 15 -hour 10 -minute 30

# 指定性别和最低分数
$ namer -xing 王 -keywords 明,轩,浩,然 -year 2024 -month 3 -day 15 -hour 10 -minute 30 -gender 1 -score 70
```
