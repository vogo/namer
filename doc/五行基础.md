# 五行基础 (金木水火土)

## 五行相生

相生即相互滋生、促进：

- 木生火 — 木为燃料，助火燃烧
- 火生土 — 火燃烧后化为灰土
- 土生金 — 土中蕴藏金属矿石
- 金生水 — 金属冷凝可生水
- 水生木 — 水滋润草木生长

## 五行相克

相克即相互制约、克制：

- 木克土 — 树木扎根破土
- 土克水 — 土筑堤坝挡水
- 水克火 — 水浇灭火焰
- 火克金 — 烈火熔化金属
- 金克木 — 金属利器伐木

## 取名中的五行应用原则

1. 名字中相邻字的五行应为 **相生** 关系，最为吉利
2. 姓的五行 → 生 → 名第一字的五行 → 生 → 名第二字的五行（连续相生最佳）
3. 名字中相邻字的五行 **相克** 则不利
4. 五行补益应以 **喜用神** 为准，而非简单补缺
