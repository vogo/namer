# 姓名阴阳平衡

## 基本原则

阴阳是中国哲学的根本概念。在姓名学中，阴阳的判定基于汉字的康熙笔画数：

- **奇数笔画 = 阳** (1, 3, 5, 7, 9, ...)
- **偶数笔画 = 阴** (2, 4, 6, 8, 10, ...)

核心理念："纯阴不发，独阳不长" — 全阴或全阳都不利。

## 三字姓名的阴阳格局

以"姓-名1-名2"的阴阳排列评判：

### 最佳（阴阳交替）
- 阳-阴-阳
- 阴-阳-阴

### 较好（有阴有阳）
- 阴-阳-阳
- 阳-阳-阴
- 阳-阴-阴
- 阴-阴-阳

### 不利（纯阴或纯阳）
- 阳-阳-阳（纯阳，过刚易折）
- 阴-阴-阴（纯阴，缺乏生机）

## 两字姓名的阴阳格局

- 阳-阴 或 阴-阳：吉
- 阳-阳 或 阴-阴：不利

## 评分建议

| 格局         | 评价 | 建议分值 |
|--------------|------|----------|
| 阴阳交替     | 最佳 | 满分     |
| 有阴有阳     | 较好 | 80%      |
| 纯阴或纯阳   | 不利 | 40%      |
